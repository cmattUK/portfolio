Horror obsessed

Growing up in the VHS era meant I was exposed to a myriad of excessive and colourful horror movies. This early doorway into darkness gave me an insatiable appetite to consume as much horror fiction as possible; which is why you’ll usually find me tackling the latest Resident Evil or rewatching Evil Dead 2 - when I should be out in the sun, like normal people!

Horror is such a huge influence on my work. This site was inspired by Remedy's Control and Stuart Gordon's From Beyond.

Recent Work

Jab Cross Hook

Contender Promotions founder, Gary Bell, designed this wonderful family-friendly card game during lockdown. I volunteered to provide both a responsive product website and a teaser trailer. This was an interesting project as it required me to create a wealth of original Photoshop graphics as well as developing my dormant After Effects skills.

Responsibilities: CSS | HTML | DESIGN | BOOTSTRAP | VIDEO

HR minisite

I was tasked with redesigning the HR section of tees.ac.uk. Although much of graphic design was dictated by mockups submitted by the client there was a huge amount of server-side code to implement. Creating the job search was definitely a highpoint; this involved building multiple filters to asynchronously (AJAX) return search results.

Responsibilities: CSS | HTML | DESIGN | JQUERY | SQL | AJAX | COLDFUSION

Student Life

With Teesside University rebranding and centralising their student facing services the client wanted a new minisite to do the same. This included some very tricky tasks such as: creating an SQL search that requested results from a Knowledge Base API using JSON objects, integrating a Chatbot, and creating animated tiles with JQUERY.

Responsibilities: CSS | HTML | DESIGN | JQUERY | JSON | SQL | COLDFUSION

Beyond the Veil

Starting as a personal blog to express my interest in horror, Beyond the Veil grew incredibly quickly. Within a few months we had a volunteer staff of writers from around the globe and we had gained a multitude of industry links meaning we were given exclusive access pre-release products. Although this was built in Squarespace, as the founder I had a slew of responsibilities including: branding/design, editing articles, writing reviews, pr, comms and video production.

Responsibilities: CSS | DESIGN | BRANDING | VIDEO
